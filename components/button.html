Button

Button triggers an event or action. They let users know what will happen next.

Examples

Basic usage

The default Strawberry theme exports button styles to get you started quickly. Import the button module to include the styles.

<script> import '@kwangure/strawberry/default/button'; let count = 0; const increment = () => count++; </script> <button on:click={increment} style="margin-right: 10px;"> Increment Counter </button> {count}

Primary button

Add the .br-primary class to a button to give it an accent color that puts emphasis on it.

<script> import '@kwangure/strawberry/default/button'; </script> <button class='br-primary'> I'm a primary button </button> <button> I'm a plain old button </button>

Alternative buttons

Generally, only the <button/> element should be used for actions. In some cases though, what seems like an action to the user is actually a navigation in the app or a general click. Use the .br-button class on non-button elements to give them a button appearance.

<script> import '@kwangure/strawberry/default/button'; </script> <a href='#/do/thing' class='br-button' role='button'> I'm an anchor tag </a> <div class='br-button' role='button'> I'm a div tag </div>

Primary button

Add the .br-ignore class to exclude a button from Strawberry styling.

<script> import '@kwangure/strawberry/default/button'; </script> <button> I have Strawberry styles </button> <button class='br-ignore'> I have UserAgent styles </button>
