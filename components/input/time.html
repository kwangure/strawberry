Accessibility

You should always include the label text for the input for the benefit of screen-reader users. You may use custom-properties to visually hide it if you must. For example: 

<div class="element">
    <Time>
        <svelte:fragment slot="label">
            Delivery time
        </svelte:fragment>
    </Time>
</div>

<style>
    .element {
        --br-input-time-label-position: absolute;
        --br-input-time-label-height: 1px;
        --br-input-time-label-width: 1px;
        --br-input-time-label-overflow: hidden;
    }
</style>
